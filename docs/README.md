# Reclio docs

Docusaurus-powered documentation for Reclio.

## Run locally

```bash
cd docs
npm install
npm start          # http://localhost:3000
```

## Build

```bash
npm run build
npm run serve      # preview the production build
```

## Deploy to GitHub Pages

```bash
GIT_USER=<your-github-user> npm run deploy
```

The site publishes to the `gh-pages` branch on the repo. Point your
GitHub Pages settings at that branch.
